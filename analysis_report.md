# Complete Sales Data Analysis & Visualization

## Project Overview
This project performs a complete data analysis workflow using Python.
It includes data loading, cleaning, analysis, and visualization to
extract meaningful business insights.

---

## Tools Used
- Python
- Pandas
- Matplotlib

---

## Data Cleaning
- Removed missing values
- Removed duplicate records
- Ensured correct data types

---

## Key Metrics
- Total Sales calculated
- Average Sales per transaction
- Product-wise revenue distribution

---

## Visual Insights

### Bar Chart
Shows comparison of total sales across products.

### Line Chart
Displays sales trend over time.

### Pie Chart
Illustrates revenue contribution by each product.

---

## Business Insights
- Certain products contribute significantly more to total revenue
- Sales trends help identify peak periods
- Visualization makes insights easier to understand

---

## Conclusion
This project demonstrates a complete data analysis pipeline and shows
how visualizations help convert raw data into business decisions.
